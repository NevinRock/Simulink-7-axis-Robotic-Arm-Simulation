📌 About This Project

This repository provides a 7 Degrees-of-Freedom (7-DOF) robotic arm simulation built in Simulink.

It is mainly intended for:

🎓 Learning robotic arm modeling
🧠 Understanding kinematics and control
⚙️ Practicing Simulink system design
✨ What You Will Learn

By following the tutorial, you will learn:

Forward Kinematics (FK)
Basic robot modeling in Simulink
Joint control and motion simulation
How a multi-DOF robotic arm works

🚀 How to Use
🥇 1. Clone the repository
git clone https://github.com/NevinRock/Simulink-7-axis-Robotic-Arm-Simulation.git
🥇 2. Open MATLAB

Open the project folder and add it to path:

addpath(genpath(pwd))
🥇 3. Open the Simulink model
open('your_model.slx')

👉 Replace with the actual .slx file name in this repo.

🥇 4. Run simulation

Click the Run ▶ button in Simulink.
