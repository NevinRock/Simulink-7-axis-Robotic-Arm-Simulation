## 📌 项目简介

本项目实现了一个 7 自由度（7-DOF）机械臂的仿真系统，基于 MATLAB 与 Simulink 环境搭建，主要用于：

机械臂运动学 / 动力学建模学习
控制系统设计与仿真验证
课程配套实践与复现

项目适合作为机器人入门与进阶学习案例，尤其适合：

自动化 / 机器人 / 控制工程方向学生
MATLAB & Simulink 学习者
机械臂建模与控制初学者

## 🧠 项目内容

本项目围绕 7 自由度机械臂仿真系统，涵盖从建模到控制的完整流程。

### 📦 模型与工程文件
7轴机械臂 Simulink 主模型（.slx）
MATLAB 辅助计算脚本（如参数计算、初始化等）
控制系统模块（如 PID 控制、轨迹跟踪等）
机械臂结构建模（关节与连杆定义）
### ⚙️ 核心建模与算法
1. 运动学建模
正运动学（Forward Kinematics）
逆运动学（Inverse Kinematics）
2. 动力学建模
关节力矩计算
惯性与耦合效应建模
3. 控制系统设计
误差反馈控制（Feedback Control）
PID 控制器设计与调参
轨迹跟踪控制（Trajectory Tracking）
4. 系统优化与仿真
控制参数优化
仿真稳定性与精度调整
