🦾 Simulink 7-DOF Robotic Arm Simulation

基于 MATLAB / Simulink 的 7 自由度机械臂仿真项目
配套课程学习 + 控制与建模实践

📌 项目简介

本项目实现了一个 7 自由度（7-DOF）机械臂的仿真系统，基于 MATLAB 与 Simulink 环境搭建，主要用于：

机械臂运动学 / 动力学建模学习
控制系统设计与仿真验证
课程配套实践与复现

项目适合作为机器人入门与进阶学习案例，尤其适合：

自动化 / 机器人 / 控制工程方向学生
MATLAB & Simulink 学习者
机械臂建模与控制初学者

📚 本仓库 = 课程的工程实现版本 / 实践补充

🧠 项目内容

项目主要包括：

7轴机械臂 Simulink 模型（.slx）
MATLAB 辅助计算脚本（如有）
仿真控制模块（如 PID / 轨迹控制等）
机械臂结构建模

通常类似项目会包含：

运动学建模（正解 / 逆解）
动力学建模（关节力矩 / 惯性）
控制系统（误差反馈、控制器）
这些都是机器人仿真的核心组成部分
🚀 如何运行
1️⃣ 环境要求
MATLAB（推荐 R2021a 及以上）
Simulink
（可选）Simscape / Simscape Multibody
2️⃣ 运行步骤
1. 克隆仓库
git clone https://github.com/NevinRock/Simulink-7-axis-Robotic-Arm-Simulation

2. 打开 MATLAB
3. 将项目目录加入 MATLAB 路径

4. 打开 .slx 文件
5. 点击 Run 运行仿真
